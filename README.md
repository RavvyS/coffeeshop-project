# ☕ My Cafe - Coffee Shop Website

A simple, beautiful coffee shop website built with Next.js and TypeScript.

## 🚀 Getting Started

1. **Start the development server:**
   ```bash
   npm run dev
   ```

2. **Open your browser:**
   Go to [http://localhost:3000](http://localhost:3000)

## ✨ What's Included

- ✅ Beautiful homepage with coffee shop branding
- ✅ Menu page with sample items
- ✅ Responsive design (works on phone and desktop)
- ✅ Coffee shop color theme
- ✅ Modern, clean design

## 🎨 Customization

### Change Coffee Shop Name
Edit `src/components/Header.tsx` and change "My Cafe" to your shop name.

### Change Colors
Edit `tailwind.config.js` and modify the coffee/cream color values.

### Add Menu Items
Edit `src/app/menu/page.tsx` and add your items to the `menuItems` array.

## 📁 Project Structure

```
src/
├── app/           # Your website pages
├── components/    # Reusable parts (Header, etc.)
└── globals.css    # Styling
```

## 🛠️ Commands

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run start    # Start production server
```
